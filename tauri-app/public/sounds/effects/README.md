# 音效文件说明

此文件夹用于存放特效音文件。

## 需要的音效文件

请将以下音效文件放置在此目录下：

### 人声类
1. `applause.mp3` - 掌声（约2秒）
2. `laugh.mp3` - 笑声（约1.5秒）
3. `cheer.mp3` - 欢呼（约2秒）
4. `gasp.mp3` - 惊讶/倒吸气（约0.5秒）

### 环境类
5. `doorbell.mp3` - 门铃（约1秒）
6. `phone-ring.mp3` - 电话铃声（约2秒）
7. `knock.mp3` - 敲门（约1秒）

### 提示类
8. `notification.mp3` - 通知音（约0.5秒）
9. `success.mp3` - 成功提示音（约1秒）
10. `warning.mp3` - 警告音（约1秒）

## 音效获取来源

### 推荐免费资源（CC0或CC-BY授权）

1. **Freesound.org** (https://freesound.org/)
   - 搜索关键词：applause, laugh, cheer, doorbell等
   - 筛选：选择CC0或CC-BY授权
   - 格式：下载MP3格式

2. **Zapsplat.com** (https://www.zapsplat.com/)
   - 免费注册后可下载
   - 个人项目免费使用

3. **Mixkit.co** (https://mixkit.co/free-sound-effects/)
   - 完全免费商用
   - 高质量音效

4. **Pixabay** (https://pixabay.com/sound-effects/)
   - CC0授权，无需署名
   - 免费下载

## 音效要求

- **格式**: MP3
- **音质**: 128kbps 或更高
- **文件大小**: 单个文件 < 200KB
- **音量**: 标准化处理，避免过大或过小
- **采样率**: 44.1kHz 或 48kHz

## 快速开始（临时测试）

如果只是想快速测试功能，可以：
1. 使用任意短音频文件重命名为上述文件名
2. 或者使用在线TTS生成简短提示音
3. 后续再替换为专业音效
