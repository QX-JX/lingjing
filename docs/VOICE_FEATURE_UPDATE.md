## 🎭 当前多发音人功能使用方式

### 基本使用

1. **选中文本**
   - 在编辑器中选中要指定发音人的文本段落
   
2. **点击工具栏"多发音人"按钮**
   - 如果未选中文本，会提示："请先选中要指定发音人的文本"
   - 如果已选中文本，会弹出发音人下拉框

3. **选择发音人**
   - 从 12 位发音人中选择一个
   - 点击后，选中的文本会被包裹在 `<voice>` 标签中

### 示例效果

**操作前**：
```
这是一段普通文本。
```

**选中"一段普通文本"，选择"晓辰 (女)"**：
```
这是<voice voice_id="xiaochen" voice_name="晓辰 (女)" voice_avatar="晓">一段普通文本</voice>。
```

### 多段发音人组合

可以为不同文本段落指定不同发音人：

```
<voice voice_id="zhiwei" voice_name="云希 (男)" voice_avatar="云">欢迎收听今日新闻。</voice>
<voice voice_id="xiaochen" voice_name="晓辰 (女)" voice_avatar="晓">下面是详细报道。</voice>
<voice voice_id="yunfeng" voice_name="云枫 (男)" voice_avatar="云">本台记者报道。</voice>
```

生成音频时，系统会自动识别不同的发音人标记，使用相应的声音朗读对应的文本。
