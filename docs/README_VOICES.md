# 发音人使用说明

## 可用的发音人

本应用支持以下 **6 个**由 edge-tts 原生支持的中文发音人：

### 男性发音人（3个）
1. **云希** (活泼灵动) - 阳光、活泼、富有朝气，声音像青少年或充满活力的年轻人。适合动画旁白、短视频讲解、小说中的少年角色、轻松的教学视频
2. **云野** (专业稳重) - 雄浑、专业、有磁性，接近传统广播电台的新闻主播。适合新闻播报、纪录片旁白、企业宣传片、严肃的学术讲座、时政解说
3. **云健** (激情澎湃) - 语速较快，充满力量感和运动感，声音亢奋极具感染力。适合体育赛事解说、竞技类游戏解说、促销广告、短视频中的"咆哮式"旁白

### 女性发音人（3个）
1. **晓晓** (全能情感) - 微软的"当家花旦"，音质温婉、亲切、自然，支持多种情绪（开心、悲伤、愤怒、恐惧、低语等）。适合有声小说、影视解说、情感类电台、客服机器人、长文本阅读
2. **晓伊** (温柔甜美) - 声音清脆、甜美、有礼貌，像专业的服务人员或知心大姐姐。适合在线客服、语音助理、儿童故事、商场广播、生活贴士提醒
3. **云霞** (可爱童真) - 稚嫩、纯真、可爱，典型的儿童声线。适合幼儿教育、儿童绘本朗读、动画片低幼角色、萌宠视频配音

## 为什么只有6个发音人？

edge-tts 是基于微软 Azure TTS 的免费服务，目前只支持这 6 个中文大陆发音人。其他您可能在网上看到的发音人（如云枫、云泽、晓辰等）需要付费的 Azure TTS API 才能使用。

## 如何选择合适的发音人？

### 根据内容类型选择

- **动画旁白/短视频/少年角色** → 云希（活泼灵动）
- **新闻播报/纪录片/企业宣传** → 云野（专业稳重）
- **体育解说/游戏解说/促销广告** → 云健（激情澎湃）
- **有声小说/影视解说/情感电台** → 晓晓（全能情感）
- **在线客服/语音助理/儿童故事** → 晓伊（温柔甜美）
- **幼儿教育/儿童绘本/动画片** → 云霞（可爱童真）

### 根据性别选择

- **男声**：云希、云野、云健
- **女声**：晓晓、晓伊、云霞

## 测试工具

如果您想查看所有可用的发音人，可以运行：

```bash
python python/list_available_voices.py
```

这将列出所有 edge-tts 支持的中文发音人（包括港台地区）。
