## 使用说明

### 在 Python 中使用

```python
import edge_tts
import asyncio

async def test_voice():
    # 使用云希（男性）
    communicate = edge_tts.Communicate("你好，这是一段测试文本", "zh-CN-YunxiNeural")
    await communicate.save("output.mp3")
    
    # 使用晓晓（女性）
    communicate = edge_tts.Communicate("你好，这是一段测试文本", "zh-CN-XiaoxiaoNeural")
    await communicate.save("output2.mp3")

asyncio.run(test_voice())
```

## 注意事项

1. **免费限制**: edge-tts 免费版本支持所有上述发音人
2. **网络要求**: 需要稳定的网络连接，因为需要连接到 Microsoft 服务器
3. **API 稳定性**: edge-tts 使用非官方 API，可能受到 Microsoft 政策变化影响
4. **方言支持**: 目前仅支持东北话和陕西话两种方言
5. **地区差异**: 香港和台湾的发音人使用繁体中文，语音风格可能有所不同
